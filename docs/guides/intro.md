## Introduction to Discord.Addons

These are packages to complement the Discord.Net CommandService and do much heavy lifting, so that it becomes a lot simpler for you to get it done.

Available packages:
* [Discord.Addons.MpGame](xref:Addons.MpGame.Introduction): Easily create turn-based games between Discord users.
