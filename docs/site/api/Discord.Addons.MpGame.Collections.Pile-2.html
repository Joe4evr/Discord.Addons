Class Pile<TCard, TWrapper>

Base type to represent a pile of objects, specifically for use in card games.

BrowseAndTake(Func<ImmutableDictionary<Int32, TCard>, Task<Int32[]>>, Func<TCard, Boolean>, Func<ImmutableArray<TCard>, IEnumerable<TCard>>)

Browse for and take one or more cards from the pile in a single operation.
Requires CanBrowse and CanTake.

Declaration
public Task<ImmutableArray<TCard>> BrowseAndTake(Func<ImmutableDictionary<int, TCard>, Task<int[]>> selector, Func<TCard, bool> filter = null, Func<ImmutableArray<TCard>, IEnumerable<TCard>> shuffleFunc = null)
Parameters
Type Name Description
Func<System.Collections.Immutable.ImmutableDictionary<Int32, TCard>, Task<Int32[]>> selector

A function that returns an array of the indeces of the desired cards.
The key of each pair is the index of that card.
Returning a null or empty array is considered choosing nothing and will return an empty array.
Func<TCard, Boolean> filter

An optional function to filter to cards that can be taken.
Func<ImmutableArray<TCard>, IEnumerable<TCard>> shuffleFunc

An optional function to reshuffle the pile after taking the selected cards.
If provided, requires CanShuffle.
Returns
Type Description
Task<ImmutableArray<TCard>>

The cards at the chosen indeces, or an empty array if it was chosen to not take any.
Examples
    // An effect was used that allows the user to
    // search their deck for a number of red cards
    var picked = await deck.BrowseAndTake(async cards =>
    {
        // Example: Call a method that shows
        // the available options to the user
        await ShowUser(cards);
        // Example: Call a method that waits
        // for the user to make a decision, with 'num'
        // being the max amount of cards they can choose
        return await UserInput(num);
    },
    // Only pass in the red cards
    filter: c => c.Color == CardColor.Red,
    // Shuffle the pile afterwards
    // with some custom function
    shuffleFunc: cards => cards.ShuffleItems());
    // Add the chosen cards to the user's hand:
    foreach (var card in picked)
        player.AddToHand(card);
Exceptions
Type Condition
System.InvalidOperationException

The pile does not allow browsing AND taking
-OR-
The sequence produced from shuffleFunc was null.
System.ArgumentNullException

selector was null.
System.IndexOutOfRangeException

One or more of the selected indices was not within the provided indices.
